# 🧠 Quiz Console App

A simple **console-based Quiz Application** built in **Java** to practice Object-Oriented Programming (OOP) concepts. The application presents multiple-choice questions, accepts user input, and displays the final score at the end of the quiz.

---

## 📌 Features

- 📖 Displays multiple-choice questions
- ⌨️ Accepts answers from the user
- ✅ Compares user responses with correct answers
- 📊 Calculates and displays the final score
- 🏗️ Built using Object-Oriented Programming principles

---

## 🛠️ Technologies Used

- Java
- OOP Concepts
- Arrays
- Constructors
- Encapsulation (Getters & Setters)

## 📖 Class Overview

### 🔹 Main.java
- Entry point of the application.
- Creates the `QuestionService` object.
- Starts the quiz.
- Displays the final score.

### 🔹 Question.java
Represents a quiz question.

Each object stores:

- Question ID
- Question Statement
- Four Options
- Correct Answer

Uses:
- Constructor
- Getters & Setters
- `toString()` Method

### 🔹 QuestionService.java

Handles the complete quiz logic.

Responsibilities:
- Stores questions in an array
- Displays each question
- Takes user input
- Stores user answers
- Evaluates answers
- Calculates the final score
