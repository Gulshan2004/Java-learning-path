# 🧮 Arithmetic Calculator (Java)

A simple **Console-Based Arithmetic Calculator** built using **Core Java**.  
This project demonstrates the use of **conditional statements (`if-else`)**, **loops (`while`)**, variables, operators, and basic arithmetic operations.

---

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ✅ Handles division by zero
- 🔁 Uses a `while` loop for repeated execution
- 💬 Displays a thank-you message after execution

---

## 🛠️ Technologies Used

- Java
- Core Java
- VS Code / Any Java IDE

## 🚀 How It Works

1. Initializes two numbers.
2. Selects an arithmetic operator.
3. Performs the corresponding calculation using `if-else` statements.
4. Checks for division by zero before dividing.
5. Prints the result.
6. Ends the program with a thank-you message.
